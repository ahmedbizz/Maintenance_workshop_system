# 🔧 Maintenance Workshop Management System

A scalable backend system designed to manage workshop operations, including maintenance requests, technicians, customers, and notifications.

---

## 📌 Overview
This project is a complete Workshop Management System built using ASP.NET Core.  
It helps manage repair processes, track devices, assign technicians, and handle workflow efficiently.

The system is designed with clean architecture principles to ensure scalability, maintainability, and performance.

---

## 🎯 Features

- 🔐 Authentication & Authorization (Role-Based)
- 👨‍🔧 Technician & Engineer Management
- 📦 Device Receiving & Tracking
- 🔔 Notification System
- 🛠️ Maintenance Workflow Management
- 🧾 Request & Status Tracking
- 🏢 Multi-role system (Admin, Technician, Receiver, Engineer)
- 📊 Structured database with optimized queries

---

## 🧰 Tech Stack

- ASP.NET Core MVC / Web API
- Entity Framework Core
- SQL Server
- Repository Pattern
- Unit of Work Pattern
- Clean Architecture

---

## 🏗️ Architecture

This project follows:

- Clean Architecture
- Repository & Unit of Work Pattern


##🔑 Roles in System

- Admin → Full control

- Technician → Handles maintenance tasks

- Engineer → Reviews and supervises work

- Receiver → Receives devices and creates requests
